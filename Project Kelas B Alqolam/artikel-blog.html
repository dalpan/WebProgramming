REMOTE SMART HAFIZ ERROR SAAT KLIK JUZ 30? BEGINI CARANYA

November 23, 2017

Jika Smart Hafiz tiba-tiba error saat Anda ingin memutar juz 30 menggunakan remote control, Anda bisa mencoba mempebaikinya dnegan cara sebagai berikut:

  1. Download file berikut ini: Update tombol Juz 30
  2. Jika sudah, ekstrak file dalam bentuk .rar tersebut
  3. Save folder yang telah diekstrak ke dalam usb penyimpanan
  4. Setelah folder dipindahkan ke USB, masukkan USB ke Smart Hafiz
  5. Buka menu “Others”
  6. Pilih “My File”
  7. Pilih USB dan cari file yang sudah diekstrak tadi
  8. Klik “enter”
  9. Akan ada konfirmasi untuk upgrade firmware, dan klik “Ya”
  10. Tunggu prosesnya hingga Smart Hafiz restrart kembali
  11. Coba pencet tombol Juz 30 pada remote control

Jika berhasil, Smart Hafiz akan merespon dengan baik perintah tombol dan menampilkan pilihan murottal Juz 30 dengan pilihan “Farid”, “Mila”, atau “Maqamat”. Dan Anda tinggal memilih ingin mendengarkan murottal yang mana.

Selamat mencoba!
